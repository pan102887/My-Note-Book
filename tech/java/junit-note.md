# JUnit 简述
## JUnit 5和之前版本的区别
- 整合了更多的功能，包含了更多的模块，包含了以下内容:
  - *JUnit Platform*
  - *JUnit Jupiter*
  - *JUnit Vintage*

## 支持的java版本
JUnit 5支持java 8以及更高版本
