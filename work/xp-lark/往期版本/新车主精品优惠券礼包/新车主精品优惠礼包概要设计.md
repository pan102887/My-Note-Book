# 功能描述
## 券包所需支持的功能
1. 派发限制
   1. 根据派发状态控制券包是否可以派发
   2. 可限制对同一用户派发次数
2. 优惠券计划配置
   1. 一个券包可配置单个或多个优惠券计划
3. 推送消息
   1. 对每个券包配置单独短信模板和APP内容模板,(短信模板支持配置短链接)
   2. 不同与普通优惠券派发的一张优惠券发送一次消息，优惠券包中的多张优惠券派发只推送一次消息。（当优惠券包中所有优惠券均派发失败时，不对用户发送消息）
4. 派券方式
   1. 主动派券（对一个用户派发一个券包，对一个用户派发多个券包，对多个用户派发一个券包，对多个用户派发多个券包）；
   2. 外系统调用派券接口进行派券

# 问题点
## 券包相关
1. `优惠券名称套用优惠券包名称，优惠券金额套用券包内优惠金额总和`如何理解
2. 券包受否需要主动派发功能
3. 券包的派发是否需要保证包内所有券都可派发，(券包中若有部分券的状态为不可派发, 是否可以指派发部分券)，若包中部分券派发失败，是否终止派发券包`只对券包内可派发的优惠券进行派发`
4. 券包是否可对用户派发多次，或者是否需要限制券包的派发次数
5. 券包自动派发的时间点是否需要可自定义（目前需求中）
